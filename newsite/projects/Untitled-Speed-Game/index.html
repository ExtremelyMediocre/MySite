Untitled Speed Game

A fast-paced movement-focused game inspired by arcade speed-runs and tight, responsive controls.

Untitled Speed Game is an experimental project exploring momentum-driven gameplay, tight feedback loops, and a pure focus on player movement. The goal is to create something that feels immediately responsive and satisfying, where mastering the controls leads to faster times and smoother runs.

This project is currently early in development. Core mechanics, UI, visual direction and level design are still evolving. I’m iterating quickly and treating this as a learning exercise in game feel, polish, and rapid prototyping.

Technologies & Focus Areas

  • Fast movement & traversal mechanics
  • Player feedback & game feel
  • Iteration-driven prototyping
  • Level flow & replayability
