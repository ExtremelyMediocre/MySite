3DPrinted-ESP32-Lamp

A custom-designed 3D printed smart lamp powered by an ESP32 microcontroller and addressable LEDs — combining hardware tinkering, enclosure design, and interactive lighting.

About the Project

This lamp is built around an ESP32 board driving addressable LEDs inside a fully 3D-printed enclosure. The project explores enclosure design, heat management, diffused lighting, and microcontroller-driven effects.

Future plans include Wi-Fi control, sound-reactive lighting, and a more refined internal design for easier assembly.
